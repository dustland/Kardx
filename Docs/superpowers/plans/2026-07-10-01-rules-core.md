# OpenCards Rules Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a deterministic, headless GDScript rules engine for complete shared-frontline matches.

**Architecture:** Pure `RefCounted` state and rule classes own all game state. `MatchController.submit_action()` is the only mutation gateway and returns ordered events plus a copied snapshot. UI and AI are deliberately absent from this plan.

**Tech Stack:** Godot 4.7, typed GDScript, JSON, custom headless test runner, seeded `RandomNumberGenerator`.

## Global Constraints

- Use `Credit`, never `Kredit`, in OpenCards UI, JSON, code, tests, and documentation.
- A deck is 39 playable cards plus one Headquarters; copy limits are Standard 4, Limited 3, Special 2, Elite 1.
- Each Support Line has one Headquarters plus four unit slots; the shared Frontline has five unit slots and one controller.
- Hand size is nine, Headquarters defense is 20, and normal Credit slots increase to 12.
- Rule code must not depend on `Node`, `Control`, scenes, global randomness, or frame timing.
- All accepted actions and AI choices must be deterministic for the same seed and state.
- No KARDS card names, text, artwork, logos, or visual trade dress may be copied.

```gdscript
class_name GameAction
extends RefCounted

var type: String
var actor_id: String
var source_id: String
var target_ids: Array[String]
var payload: Dictionary
var expected_sequence: int

static func create(action_type: String, actor: String, source: String = "", targets: Array[String] = [], data: Dictionary = {}, sequence: int = 0) -> GameAction:
    var action := GameAction.new()
    action.type = action_type
    action.actor_id = actor

    action.target_ids = targets.duplicate()
    action.payload = data.duplicate(true)
    action.expected_sequence = sequence
    return action
```

```gdscript
func drain_queue() -> Dictionary:
    var emitted: Array[Dictionary] = []
    while not queue.is_empty():
        if emitted.size() >= GameConstants.MAX_EFFECT_EVENTS:
            return {"valid": false, "code": "effect_limit", "events": emitted}
        var effect: Dictionary = queue.pop_front()
        emitted.append(_apply_effect(effect))
    return {"valid": true, "events": emitted}
```
